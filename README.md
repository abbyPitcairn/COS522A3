# COS522A3

This is a simple neural network program for COS 522, Assignment 3. 

### How to Run:

Make sure you are in the directory 'COS522A3'. Install requirements, then run the main script.

You can use the following commands:

```
cd COS522A3
python requirements.txt
python A3.py
```

### Data:

The data here is purely for example. Each row has five numerical features and two categorical, and each has one multi-class label [A, B, C]. For categorical features, we employ one-hot encoding to avoid learning false correlations. 

**Example Data:**

| num1 | num2 | num3 | num4 | num5 | cat1 | cat2 | label |
|---|---|---|---|---|---|---|---|
|0 | -2 | 276 | 22 | -190 | Green | Medium | B |
| 68 | 39 | 196 | 996 | -118 | Blue | Medium | A |
| 3 | -76 | 69 | 564 | -206 | Green | Small | C |
